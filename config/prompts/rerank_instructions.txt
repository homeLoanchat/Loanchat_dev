# 리랭커 지침
# - 후보 문서 정렬 기준(정확성, 최신성, 출처 신뢰도)을 정의하세요.
# - 스코어를 [0,1] 범위로 정규화하는 규칙과 임계치를 기록하세요.
# - 다국어/요약 결과가 필요한 경우 추가 지침을 명시하세요.

[목적]
검색 또는 RAG 결과의 신뢰도를 평가해 재정렬한다.

[정렬 기준]
1. 정책·요건·한도·금리 등 핵심 금융정보를 포함한 문서를 우선한다.
2. 금융위원회·금융감독원·한국은행 등 공공기관 문서의 점수를 높인다.
3. 발표일이 최근일수록 가중치를 높인다.
4. 중복 내용은 상위 1개만 유지한다.

[스코어 처리]
- 모델 점수를 0~1 범위로 정규화한다.
- 0.75 이상은 높은 신뢰도로 간주한다.
- 0.4 미만은 참고용으로만 사용한다.

[언어 처리]
- 비한국어 문서는 요약 후 한국어로 변환한다.
- 요약은 핵심 문장 3~5개로 구성한다.
